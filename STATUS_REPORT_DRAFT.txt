======= 
1. Research Idea for implementation: 
Our system, APT-Guard, is a Deep Q-Learning (DQL) based security extension for PostgreSQL 18.3 that detects multi-stage Advanced Persistent Threats by correlating database session history with real-time Linux process-level telemetry.

Key Supporting Points:
- Uses a Deep Q-Network (DQN) to move beyond single-query anomaly detection by analyzing temporal sequences of database events (Recon -> Escalation -> Exfiltration).
- Bridges the database and the OS by mapping SQL session IDs to Linux Process Descriptors (task_struct) for deep behavioral forensics.
- Provides Adaptive Defense by automatically learning optimal responses (Alert, Rate-limit, or Block) to neutralize threats without disrupting benign traffic.

2. The reference research paper link from where that idea is born: 
M. A. Hossain, “Deep Q-learning intrusion detection system (DQ-IDS): A novel reinforcement learning approach for adaptive and self-learning cybersecurity,” ICT Express, vol. 11, pp. 875–880, 2025.
